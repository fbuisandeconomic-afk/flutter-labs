# МИР - РЕЦЕПТОВ

**Краткое описание:**  
Приложение для управления и поиска рецептов. Позволяет пользователю просматривать список блюд, удалять их из истории, а также искать новые рецепты по названию с отображением названия, текста рецепта и изображения.

---

## Основные функции

- **Просмотр истории рецептов** — отображение списка ранее добавленных или просмотренных блюд (Блюдо 1, Блюдо 2 и т.д.).
- **Удаление рецептов** — возможность удалить любое блюдо из истории через кнопку корзины.
- **Поиск рецептов** — поле ввода для поиска по названию блюда.
- **Отображение деталей рецепта** — после поиска показывается:
  - Название блюда
  - Текст рецепта
  - Изображение блюда
- **Навигация между экранами** — переключение между главным экраном (поиск) и экраном истории.

---

## Структура экранов

### 1. Экран «История» (`History Screen`)
- Заголовок: **МИР - РЕЦЕПТОВ**
- Список элементов:
  - Каждый элемент — название блюда (например, «Блюдо 1», «Блюдо 2»...)
  - Справа от каждого — кнопка удаления (иконка корзины с крестиком)
  - После пятого элемента — многоточие (`......`) indicating more items exist
- Нижняя навигация:
  - Кнопка **«Главная»** (неактивна / белая)
  - Кнопка **«История»** (активна / серая)

> *Примечание: В текущей версии данные статичны — в реальной реализации здесь будет динамический список из базы данных или локального хранилища.*

---

### 2. Экран «Главная» (`Main Screen`)
- Заголовок: **МИР - РЕЦЕПТОВ**
- Поле поиска:
  - Плейсхолдер: *«Введите блюдо для поиска»*
  - Иконка лупы справа
- Результаты поиска (шаблон):
  - `{Название блюда}`
  - `{Рецепт}`
  - `{Картинка}`
- Нижняя навигация:
  - Кнопка **«Главная»** (активна / серая)
  - Кнопка **«История»** (неактивна / белая)

> *Примечание: Это шаблонный экран — в реальности результаты будут подгружаться dynamically при вводе запроса.*

---

## Навигация

- Переключение между экранами осуществляется через нижнюю панель навигации:
  - **«Главная»** → открывает экран поиска рецептов
  - **«История»** → открывает экран со списком сохранённых/просмотренных блюд
- Активный экран выделяется серым фоном кнопки.
- Переход мгновенный, без анимаций (в текущем макете).
